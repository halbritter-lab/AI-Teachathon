## Before the Event

Everything you need to get ready:

- [Git Cheat Sheet](https://education.github.com/git-cheat-sheet-education.pdf) (GitHub Education) - One-page PDF of the most common Git commands
- [GitHub Skills](https://skills.github.com/) - Free interactive courses to practice Git and GitHub in your browser
- [Oh Shit, Git!?!](https://ohshitgit.com/) - Plain-language fixes for common Git mistakes (you will need this)
- [Pro Git Book](https://git-scm.com/book/en/v2) - Free, comprehensive reference if you want to go deeper
- [VS Code Documentation](https://code.visualstudio.com/docs) - Official docs for keyboard shortcuts, extensions, and settings
- [Setup Guide](/setup) - Our own step-by-step installation instructions

## During the Event

Useful while you're working through the workshop:

- [Hands-On Guide](/hands-on) - The workshop exercises you'll follow along with
- [AI Tools Overview](/ai-tools) - Comparison of all the AI tools with pricing and recommendations
- [GitHub Copilot Documentation](https://docs.github.com/en/copilot) - How to use Copilot's autocomplete, chat, and inline suggestions
- [Claude Code Documentation](https://docs.anthropic.com/en/docs/claude-code) - Guide for the CLI agent included with Claude Pro
- [Cursor Editor](https://cursor.sh/) - AI-powered code editor (VS Code fork with built-in AI)
- [GitHub Desktop](https://desktop.github.com/) - Visual Git client if you prefer clicking over typing commands

## After the Event

Keep learning after the workshop ends:

- [GitHub Learn](https://learn.github.com/) - GitHub's unified learning platform with personalized paths
- [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/) - Free bite-sized courses on AI topics (30-60 min each)
- [fast.ai Practical Deep Learning](https://course.fast.ai/) - Free full course on deep learning, beginner-friendly
- [The Missing Semester](https://missing.csail.mit.edu/) (MIT) - Shell, Git, editors, and other tools your CS classes skipped
- [Codecademy: Learn Git](https://www.codecademy.com/learn/learn-git) - Interactive Git tutorial with in-browser exercises

## API Documentation

For participants working with AI APIs programmatically:

- [Anthropic API Docs](https://docs.anthropic.com/) - Claude models, tool use, and prompt engineering guides
